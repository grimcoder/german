#### Grammar Focus

##### Modal Verbs
- Express ability, necessity, obligation, or permission.
- Example: "können" (can, to be able to)
  
  | Pronoun   | Conjugation |
  |-----------|-------------|
  | ich       | kann        |
  | du        | kannst      |
  | er/sie/es | kann        |
  | wir       | können      |
  | ihr       | könnt       |
  | sie/Sie   | können      |

##### The Accusative Case
- Used for the direct object of a sentence.
- Definite Article Changes:
  - **der** (masc.) -> **den**
  - **die** (fem.) -> **die**
  - **das** (neut.) -> **das**
  - **die** (plur.) -> **die**
- Example: "Ich sehe den Mann." (I see the man.)
